# 环境
### 硬件
    3台虚拟机，8G内存，500G硬盘，4核

### 软件

    操作系统： centos6.5  
    hadoop ： hadoop-2.6.0-cdh5.7.5  
    jdk： 1.8

# 搭建模式
    1. 单机模式
    2. 伪分布模式
    3. 完全分布式
    4. ha 模式
    5. faderation 模式
    6. faderation + viewfs 模式

    1.环境准备（免密，主机列表，关闭防火墙，句柄修改等）  
    2.安装jdk  
    3.解压安装包，修改配置，分发  
    4.格式化zk，namenode  
    5.启动服务
